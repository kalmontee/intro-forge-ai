# IntroForge AI

> AI-powered message generator for job seekers and professionals

## Overview

IntroForge AI is a web application that leverages artificial intelligence to help professionals craft personalized, compelling outreach messages for job applications. Whether you're reaching out to recruiters on LinkedIn, writing cover letters, or sending cold emails, IntroForge AI generates tailored content in seconds based on your background and target opportunity.

## Key Features

- **Personalized Message Generation** - Create custom messages tailored to specific roles, companies, and recipients
- **Multiple Message Types** - Support for LinkedIn outreach, email introductions, and formal cover letters
- **Real-Time Preview** - See your generated message instantly with formatting and tone analysis
- **Message Analytics** - Track word count, estimated read time, and tone assessment
- **Smart Customization** - Add context and highlight specific skills or achievements
- **Copy Templates** - Easily copy messages to clipboard

## Getting Started

```bash
# Clone the repository
git clone https://github.com/yourusername/introforge-ai.git

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## How It Works

1. **Input Your Details** - Provide your name, experience, target role, and company
2. **Select Message Type** - Choose between LinkedIn message, email, or cover letter
3. **Add Context** - Include any specific skills or achievements you want to highlight
4. **Generate** - Click generate to create your personalized message
5. **Refine & Use** - Copy or regenerate until perfect
